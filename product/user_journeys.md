# User Journeys

## J1: Erstnutzer (Mias Onboarding)
**Ziel:** Erster Catch in 10 min, will morgen wiederkommen.

1. App heruntergeladen (Instagram / Freundin gezeigt)
2. Splash → „Weiter mit Apple" (ein Tipp)
3. Standorterlaubnis: „Um Vögel in deiner Nähe catchen zu können "
4.**Karten-Screen als Default-Tab** — sieht **alle Catches der Stadt auf Mapbox**
5. Tippt Pin: „Rotkehlchen — von Ben" → Neugier (kein Profil, nur Name)
6. **Wechselt zu Catch-Tab** → sieht **großes Mikrofon-Icon** (Shazam-Ästhetik)
7. Tippt Mikrofon → Live-Tonaufnahme + Wellenform-Visualisierung
8. **Vorschlag-Card schiebt sich ein** → „Rotkehlchen 87%"
9. **Swipet die Card nach oben** (Pokéball-Throw-Geste) → haptisches Feedback
10. → nahtlos **Catch-Animation**: Fullscreen, Rotkehlchen-Illustration, Konfetti, Vogelgeräusch, „Erstes Rotkehlchen gecatcht!"
11. Artenkarte aufgedeckt, zu Sammlung (Bird-Dex) hinzugefügt
12. Zurück zur Karte — eigener Pin sichtbar **für die ganze Stadt** (alle Catches sind öffentlich)

**Erfolg:** Sie zeigt es ihrer Mitbewohnerin. "Ich habe gerade ein Rotkehlchen gecatcht!" → Downloads passieren.

## J2: Manueller Catch (Bens Flow)
1. Ben sieht Eisvogel an der Alster → App → Catch-Tab → Manuelle Suche
2. Sucht „Eisvogel", tippt an
3. GPS-Standort gesetzt
4. Bestätigt → auf Community-Karte sichtbar

## J3: Karte erkunden (Community-Discovery)
1. Mia will einen Eisvogel sehen → Karte → Artenfilter „Eisvogel"
2. **Alle Catches der Stadt sichtbar** — Hotspot: Alster + Ohlsdorfer Friedhof
3. Tippt Pin → Mini-Card: „Eisvogel — von Claudia, zuletzt vor 4 Tagen" → „Navigieren" → öffnet Karten-App

## J4: Wiederkehrender Nutzer (täglicher Loop, V1)
1. Push: „Buntspecht 400m von dir gesehen" → Karte öffnet sich mit hervorgehobenem Pin
2. Sie geht hin — hört Trommeln → Ton-ID: „Buntspecht 94%"
3. Catchted → neue Art! Fullscreen-Animation, Konfetti, Vogelgeräusch — Dex-Zähler springt hoch
4. *(V1: Im Feed Freundin Anna kommentieren — im MVP sieht sie Buntspecht-Pins auf der Karte)*
